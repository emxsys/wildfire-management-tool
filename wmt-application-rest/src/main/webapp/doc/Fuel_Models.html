Fuel Models

Get a collection of Fuel Models, including the original 13 and the standard 40 fuel models.

Resource URL
/rs/fuelmodels
Curl Example
$ curl -X GET -H 'Content-Type: plain/text' "/rs/fuelmodels"
 
Request Header
Content-Type: plain/text

Gets all the standard fuel models.

The standard 40 fuel models.

FUEL_MODEL_GROUP_UNBURNABLE String

Unburnable fuel models common to the original 13 and standard 40, such as urban, water, etc.

FuelModel
NameTypeDescription
modelNo int

The standard fuel model number compatible with LANDFIRE data sets.

@return The unique number identifying the fuel model.

modelCode String

The standard fuel model code.

@return The unique code representing the fuel model.

modelName String

The fuel type complex name.

@return The model name.

modelGroup String

The fuel type group that this model belongs to, e.g., Original 13, Standard 40, etc.

@return The model group name.

dead1HrFuelLoad visad.Real

The 1 hour dead fuel loading.

@return [kg/m2]

dead10HrFuelLoad visad.Real

The 10 hour dead fuel loading.

@return [kg/m2]

dead100HrFuelLoad visad.Real

The 100 hour dead fuel loading.

@return [kg/m2]

liveHerbFuelLoad visad.Real

The live herbaceous fuel loading.

@return [kg/m2]

liveWoodyFuelLoad visad.Real

The live woody fuel loading.

@return  [kg/m2]

dynamic boolean

The fuel type: static or dynamic.

@return True if the fuel model is dynamic.

burnable boolean

Burnable or non burnable.

@return True if the fuel model is burnable
